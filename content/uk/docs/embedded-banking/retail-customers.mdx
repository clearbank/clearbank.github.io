## Retail customers

These endpoints are used to create and update retail customers for all customer accounts used by our embedded banking partners.

To create an operating account, use the endpoints on the [Manage sterling accounts page](../gbp-accounts/manage-accounts) page.

<EndpointBlock
  type="post"
  title="Create a retail customer"
  description={CONSTANTS.postCustomersRetailDescription}
  endpoints={[
    {
      path: "/v1/customers/retail",
      version: "1.0.Retail"
    }
  ]}
/>

<EndpointBlock
  type="patch"
  title="Update an existing retail customer’s personal information"
  endpoints={[
    {
      path: "/v1/customers/retail/{customerId}",
      version: "1.0.Retail"
    }
  ]}
/>

<EndpointBlock
  type="put"
  title="Update an existing retail customer’s address"
  endpoints={[
    {

    }
  ]}
/>
