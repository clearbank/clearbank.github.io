## Endpoint lookup table

You can find links to more information on all our current endpoints in this table.
The rows are ordered alphabetically based on the path, ignoring the version number.

| Endpoint      | Description |
| ----------- |  ----------- |
| [GET /mccy/v1/Accounts](../multi-currency/manage-multi-currency-accounts#get-all-accounts-real) | Get a list of all the accounts associated with your institution, excluding virtual accounts |
| [POST /mccy/v1/Accounts](../multi-currency/manage-multi-currency-accounts#create-a-new-account-real) | Create a new account |
| [DELETE /mccy/v1/Accounts/{accountId}](../multi-currency/manage-multi-currency-accounts#close-an-existing-account-real) | Close an existing account |
| [GET /mccy/v1/Accounts/{accountId}](../multi-currency/manage-multi-currency-accounts#get-information-for-an-account-real) | Get details related to an account |
| [PATCH /mccy/v1/Accounts/{accountId}](../multi-currency/manage-multi-currency-accounts#amend-the-properties-of-an-account-real) | Amend the properties of an existing account |
| [GET /mccy/v1/Accounts/{accountId}/balances](../multi-currency/manage-multi-currency-accounts#get-the-balance-of-an-account-real) | Get the balance of an account |
| [POST /mccy/v1/Accounts/{accountId}/currencies](../multi-currency/manage-multi-currency-accounts#add-a-currency-to-an-existing-account-real) | Enable an existing account to support a new currency |
| [GET /mccy/v1/Accounts/{accountId}/transactions](../multi-currency/account-reporting#get-full-transaction-list-for-an-account) | Get a list of transactions for a real account and any virtual accounts associated with that real account |
| [POST /v1/mccy/inboundpayment/{accountUniqueId}](../multi-currency/multi-currency-payments#fund-your-account-in-the-simulation-environment) | Make an inbound payment to an account in the simulation environment |
| [POST /v1/mccy/internaltransfers](../multi-currency/multi-currency-payments#request-an-internal-transfer) | Initiate an internal transfer request |
| [POST /v1/mccy/payments](../multi-currency/multi-currency-payments#initiate-a-payment) | Initiate payments |
| [DELETE /v1/mccy/payments/{batchId}](../multi-currency/multi-currency-payments#cancel-batch-payments) | Cancel a batch of payments with a matching batch ID |
| [DELETE /v1/mccy/payments/{batchId}/{endToEndId}](../multi-currency/multi-currency-payments#cancel-a-single-payment) | Cancel a single payment within a batch, with a matching batch ID and end-to-end ID |
| [POST /mccy/v1/StatementRequests](../multi-currency/account-reporting#request-a-statement-for-all-accounts) | Request a statement for all accounts associated with your institution (as per the specified currency) |
| [POST /mccy/v1/StatementRequests/account/{iban}](../multi-currency/account-reporting#request-a-statement-for-an-account) | Request a statement for a specific account associated with your institution (as per the specified currency), excluding virtual accounts |
| [GET /mccy/v1/Transactions/{transactionId}](../multi-currency/account-reporting#get-a-transaction) | Get a transaction |
| [POST /mccy/v1/VirtualAccounts](../multi-currency/manage-multi-currency-accounts#create-an-account-virtual) | Create a new virtual account |
| [DELETE /mccy/v1/VirtualAccounts/{virtualAccountId}](../multi-currency/manage-multi-currency-accounts#close-an-account-virtual) | Close an existing virtual account |
| [GET /mccy/v1/VirtualAccounts/{virtualAccountId}](../multi-currency/manage-multi-currency-accounts#get-information-for-an-account-virtual) | Get details related to a virtual account |
| [PATCH /mccy/v1/VirtualAccounts/{virtualAccountId}](../multi-currency/manage-multi-currency-accounts#amend-the-properties-of-an-account-virtual) | Amend the properties of an existing virtual account |
| [PATCH /mccy/v1/VirtualAccounts/{virtualAccountId}/status](../multi-currency/manage-multi-currency-accounts#update-the-status-of-an-account-virtual) | Update the status of an existing virtual account |
| [GET /mccy/v1/VirtualAccounts/{virtualAccountId}/transactions](../multi-currency/account-reporting#get-transaction-list-for-a-virtual-account) | Get a list of transactions for a virtual account |
| [GET /v2/Transactions](../gbp-accounts/payment-data#get-all-payments-institution) | Get a summary of all of the transactions from the institution |
| [GET /v1/Transactions/{sortCode}/DirectCredit/Unapplied](../gbp-accounts/bacs-payment-data#get-all-unapplied-direct-credit-payments) | Return a list of all unapplied Direct Credit transactions for a given sort code |
| [GET /v1/Transactions/{sortCode}/DirectDebit/Unpaid](../gbp-accounts/bacs-payment-data#get-all-unpaid-direct-debit-payments-institution) | Return a list of all unpaid Direct Debit Transactions for a given sort code |
