## Transactions

<Callout colour="green"><h2>Transactions are coming soon</h2><p>All information on this page describes a solution that is <strong>not yet live in production and therefore could change</strong>. This includes any endpoint and webhook details. The information is being provided early to aid you in integrating with our API in our European simulation environment.</p>
</Callout>

Transaction endpoints allow you to retrieve:
- A list of transactions for a real account and any virtual accounts associated with it
- A list of transactions for a virtual account
- A single transaction

You can also access [Account statements](./statements).

<EndpointBlock
  type="get"
  title="Get full transaction list for an account"
  endpoints={[
    {
      path: "/mccy/v1/Accounts/{accountId}/transactions",
      version: "1.0.MCCY-Transactions-EU",
    }
  ]}
/>

<EndpointBlock
  type="get"
  title="Get transaction list for a virtual account"
  endpoints={[
    {
      path: "/mccy/v1/VirtualAccounts/{virtualAccountId}/transactions",
      version: "1.0.MCCY-Transactions-EU",
    }
  ]}
/>

<EndpointBlock
  type="get"
  title="Get a transaction"
  endpoints={[
    {
      path: "/mccy/v1/Transactions/{transactionId}",
      version: "1.0.MCCY-Transactions-EU",
    }
  ]}
/>
