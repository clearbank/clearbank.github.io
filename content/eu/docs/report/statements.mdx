## Account statements

Your bank statement is an official summary of financial transactions that have occurred for all european accounts within a calendar month.

Statements can be generated either for all your accounts, or for a specified real account. The statement will be provided via a webhook (refer to [Use webhooks](../api/getting-started/#use-webhooks)).

<EndpointBlock
  type="post"
  title="Request a statement for all accounts"
  endpoints={[
    {
      path: "/mccy/v1/StatementRequests",
      version: "1.0.MCCY-Statements",
      webhooks: [
        'mccy-institution-statement-webhook-v1'
      ]
    }
  ]}
/>
