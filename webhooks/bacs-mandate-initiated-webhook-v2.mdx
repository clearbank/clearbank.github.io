**This sends a webhook notification confirming that a Bacs Direct Debit Instruction has been initiated.**

A DDI has been received and set up.


#### Request body

```json
{
    "Type": "BacsMandateInitiatedV2",
    "Version": 2,
    "Payload": {...},
    "Nonce":
}
```
