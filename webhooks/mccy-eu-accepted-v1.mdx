---
title: "CBEU Cross-border Outbound Payment Accepted webhook"
version: 1
webhook: true
---

This webhook confirms that an outbound multi-currency EU payment has been accepted.

#### Request body

```json
{
    "Type": "Payments.CBEU.CrossBorder.OutboundPaymentAccepted",
    "Version": 1,
    "Payload": {...},
    "Nonce":
}
```
