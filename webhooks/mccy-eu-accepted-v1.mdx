---
title: "CBEU Cross-border Outbound Payment Accepted webhook"
version: 1
webhook: true
---

This webhook confirms that an outbound multi-currency EU payment has been accepted.

#### Request body

```json
{
    "Type": "Payments.CBEU.CrossBorder.OutboundPaymentAccepted",
    "Version": 1,
    "Payload": {...},
    "Nonce":
}
```

#### Payload
| Element                     | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `TransactionId`             | *Optional* <br /> Internal unique identifier for the transaction within the platform. <br /> Type: `string`<br /> Format: `guide` <br /> Pattern: `^[a-f0-9]{8}-[a-f0-9]{4}-4[a-f0-9]{3}-[89ab][a-f0-9]{3}-[a-f0-9]{12}$`                                                                                                                                                                                                                                                                                                                                                                                               |
| `EndToEndId`                | *Optional* <br /> End-to-end reference used to identify the payment consistently across all systems and parties involved in processing. <br /> Type: `string`<br /> Pattern: `^[0-9a-zA-Z\/\\-\\?:\\(\\)\\.,''\\+](\|[0-9a-zA-Z\/\\-\\?:\\(\\)\\.,''\\+]\|[0-9a-zA-Z\/\\-\\?:\\(\\)\\.,''\\+ ]{1,33}[0-9a-zA-Z\/\\-\\?:\\(\\)\\.,''\\+])$` <br />                                                                                                                                                                                                                                                                       |
| `InstructionId`             | *Optional* <br /> Identifier assigned to a specific instruction request used to correlate client input with downstream processing. <br /> Type: `string`<br />  Pattern: `^[0-9a-zA-Z\/\\-\\?:\\(\\)\\.,''\\+](\|[0-9a-zA-Z\/\\-\\?:\\(\\)\\.,''\\+]\|[0-9a-zA-Z\/\\-\\?:\\(\\)\\.,''\\+ ]{1,33}[0-9a-zA-Z\/\\-\\?:\\(\\)\\.,''\\+])$` <br />                                                                                                                                                                                                                                                                           |
| `Uetr`                      | *Optional* <br /> Universally Unique End-to-End Transaction Reference (UUID v4) used to track the payment across the correspondent banking network. <br /> Type: `string`<br /> Format: `guid` <br /> Pattern: `^[a-f0-9]{8}-[a-f0-9]{4}-4[a-f0-9]{3}-[89ab][a-f0-9]{3}-[a-f0-9]{12}$`                                                                                                                                                                                                                                                                                                                                  |
| `PaymentId`                 | *Optional* <br /> Internal identifier representing a payment instruction within the platform. <br /> Type: `string` <br /> Pattern: `^[a-zA-Z0-9_-]{0,50}$`                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| `InstructedCcy`             | *Optional* <br /> The currency in which the instructed amount is expressed given as a three-letter ISO 4217 currency code. <br /> Type: `string` <br /> Min. Length: `3` <br /> Max. Length: `3` <br /> Pattern: `^[A-Z]{3,3}$`                                                                                                                                                                                                                                                                                                                                                                                         |
| `InstructedAmount`          | *Optional* <br /> The payment amount expressed in the instructed currency. This represents the amount associated with the instruction side of the transaction. This amount can be either client-specified or system-derived from the settlement amount using the agreed exchange rate and applicable pricing rules. <br />  It is an array of objects with `Kind` and an `Identifier`.<br /> Type: `array`<br /><br /> `Kind`<br /> The kind of account identifier which is provided. This can be either IBAN, BBAN, AccountId or Descriptor (another kind of identifier).<br /> Type: `number`<br /> Format: `decimal` |
| `ExchangeRate`              | *Optional* <br /> The exchange rate applied to convert between the instructed amount and the settlement amount for the transaction. <br /> Type: `number` <br />  Format: `decimal`                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `InterbankSettlementCcy`    | *Optional* <br /> The currency in which settlement occurs between financial institutions given as a three-letter ISO 4217 currency code. <br /> Type: `string` <br /> Min. Length: `3` <br /> Max. Length: `3` <br /> Pattern: `^[A-Z]{3,3}$`                                                                                                                                                                                                                                                                                                                                                                           |
| `InterbankSettlementAmount` | *Optional* <br /> The amount settled between financial institutions as part of payment execution. This represents the settlement side of the transaction and may be either fixed or derived. <br /> Type: `number` <br /> Format: `decimal` <br />                                                                                                                                                                                                                                                                                                                                                                      |
| `PaymentValueDate`          | *Optional* <br /> The value date on which the payment is settled between financial institutions. <br /> Type: `string` <br /> Format: `date-time`                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
