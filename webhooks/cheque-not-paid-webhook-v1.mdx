---
title: "Cheque Not Paid webhook"
version: 1
webhook: true
---

This webhook confirms a cheque has been received and will not be paid.

#### Request body

```json
{
    "Type": "ChequeNotPaid",
    "Version": 1,                                                         
    "Payload": {...},
    "Nonce":
}
```

#### Payload
| Element                    | Description                |
|----------------------------|----------------------------|
| `NoPayReasonCode`          |**Required**<br/>Reason code to specify why the cheque has not been paid by the paying bank.<br/>Type: `string`<br/>Max. Length: `4`<br/>Min. Length: `1`<br/>Pattern: `^[0-9]*$`|
| `TransactionSetId`         |**Required**<br/>Unique Transaction Set ID used for Image Clearing System (ICS) processing.<br/>Type: `string`<br/>Max. Length: `22`<br/>Min. Length: `1`<br/>Pattern: `^[0-9]*$`|
| `CreditItemId`             |**Required**<br/>Unique Credit Item ID used for Image Clearing System (ICS) processing.<br/>Type: `string`<br/>Max. Length: `25`<br/>Min. Length: `1`|
| `Account`                  |**Required**<br/>Account details of the beneficiary associated with the cheque.<br/><br/>`SortCode`<br/>Type: `string`<br/>Max. Length: `6`<br/>Min. Length: `6`<br/>Pattern: `^[0-9]*$`<br/><br/>`AccountNumber`<br/>Type: `string`<br/>Max. Length: `8`<br/>Min. Length: `8`<br/>Pattern: `^[0-9]*$`|
| `DebitItemId`              |**Required**<br/>Unique Debit Item ID used for Image Clearing System (ICS) processing.<br/>Type: `string`<br/>Max. Length: `25`<br/>Min. Length: `1`|
| `ChequeAmount`             |**Required**<br/>Amount for the cheque debit item in the transaction, in sterling.<br/>Type: `number`<br/>Format: `decimal`|
| `CounterpartAccount`       |**Required**<br/>Account details for the cheque debit item in the transaction.<br/><br/>`SortCode`<br/>Type: `string`<br/>Max. Length: `6`<br/>Min. Length: `6`<br/>Pattern: `^[0-9]*$`<br/><br/>`AccountNumber`<br/>Type: `string`<br/>Max. Length: `8`<br/>Min. Length: `8`<br/>Pattern: `^[0-9]*$`|
| `ChequeSerialNumber`       |**Required**<br/>Cheque Serial Number for the cheque debit item in the transaction.<br/>Max. Length: `6`<br/>Min. Length: `1`<br/>Pattern: `^[0-9]*$`<br/>|
| `SwitchedSortCode`         |Account switching scenario - switched sort code of the cheque.<br/>Type: `string` or `null`<br/>Max. Length: `6`<br/>Min. Length: `6`<br/>Pattern: `^[0-9]*$`|
| `SwitchedAccountNumber`    |Account switching scenario - switched account number of the cheque.<br/>Type: `string` or `null`<br/>Max. Length: `8`<br/>Min. Length: `8`<br/>Pattern: `^[0-9]*$`|
| `MessageId`                |**Required**<br/>Unique identifier provided to ClearBank for each payment.<br/>Type: `string`<br/>Max. Length: `36`<br/>Min. Length: `1`|
| `MessageSourceId`          |ClearBank client identifier.<br/>Type: `string` or `null`<br/>Max. Length: `36`<br/>Min. Length: `0`|
| `TimestampCreated`         |**Required**<br/>Date and time when the webhook was created within ClearBank. Time in UTC.<br/>Type: `string`<br/>Format: `date-time`|
