title: "SEPA CT Inbound Payment Completed webhook"

webhook: true
---

This webhook confirms an inbound SCT payment request has completed.

#### Request body

```json
{
    "Type": "Sepa.Ct.InboundPayment.Completed",
    "Version": 1,
    "Payload": {...},
    "Nonce":
}
```
