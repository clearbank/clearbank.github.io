---
title: "SCT UK Outbound Payment Failed webhook"
version: 1
webhook: true
---

This webhook notifies that an outbound SCT UK payment has failed.

#### Request body

```json
{
    "Type":"Payments.Mccy.OutboundPaymentFailed",
    "Version": 1,
    "Payload": {...},
    "Nonce":
}
```

#### Payload
| Element                            | Description                                                                                                                                                                                                                                                                                |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `TransactionId`                    | **Required**<br /> A unique identifier assigned by ClearBank to reference the debit transaction that relates to this outbound payment. <br /> Type: `string`<br /> Min. Length: `1`<br /> Format: `UUID`<br />                                                                              |
| `MessageIdentification`            | **Required**<br /> A reference assigned by the original instructing party to uniquely identify the original message. <br /> Type: `string`<br /> Min. Length: `1`<br /> Max. Length `35` <br />                                                                       |
| `PaymentInformationIdentification` | **Required**<br /> A unique identifier assigned by the original sending party to distinguish the original payment information group. <br /> Type: `string`<br /> Min. Length: `1`<br /> Max. Length: `35`<br />                                                                            |
| `InstructionIdentification`        | **Required**<br /> A unique identifier assigned by the original instructing party to identify the original payment instruction. <br /> Type: `string`<br /> Min. Length: `1`<br /> Max. Length: `35`<br />                                                                                 |
| `EndToEndIdentification`           | **Required**<br /> A unique identifier assigned by the original initiating party to trace the payment from origin to destination. <br /> Type: `string`<br />  Min. Length: `1` <br /> Max. Length: `35` <br />                                                                            |
| `UETR`                             | **Required**<br /> A universally unique transaction reference (UETR) used to track the payment end-to-end across the payment chain. <br /> Type: `string`<br /> Format: `UUID`<br /> |
| `InstructedAmount`                 | **Required**<br /> The monetary amount that was instructed to be transferred in the payment. <br /> Type: `number` <br /> Format: `decimal`<br />                                                                                                                                          |
| `InstructedCurrency`               | **Required**<br /> The three-letter ISO currency code representing the currency of the instructed payment amount. <br /> Type: `string` <br /> Min. Length: `3` <br /> Max. Length: `3` <br /> Pattern: `^[A-Z]{3}$`                                                                       |
| `AdditionalInformation` | *Optional* <br/> A list of details regarding the failure, such as error codes or failure reason descriptions. <br/> Type: `array`, `null` <br/> Max. Items: `10` <br/> Item Type: `string` <br/> Item Max. Length: `140` |

```json
{
    "Type": "Payments.Mccy.OutboundPaymentFailed",
    "Version": 1,
    "Payload": {

        "MessageIdentification":"MSGID-1234-DEC25-MYREF",

        "InstructionIdentification":"INSTID-4771-MYID",
        "EndToEndIdentification":"E2EREF-BB44-MYID",

        "InstructedCurrency":"EUR",

            "UETR already in use"
        ]
    },

}
```

```json
{
    "Nonce": 961306540
}
```
