title: "Bacs Debit Contra Received webhook"

---

**This sends a webhook notification confirming a Bacs Debit Contra has been received.**

This webhook notifies a direct debit originator (ClearBank or a
customer), that an account is being credited against the debit on the
originator's account.

#### Request body

```json
{
    "Type": "BacsDebitContraReceivedV3",
    "Version": 3,
    "Payload": {...},
    "Nonce":
}
```
