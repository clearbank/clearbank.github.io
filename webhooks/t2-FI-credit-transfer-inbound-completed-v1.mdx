This webhook confirms a T2 financial institution credit transfer inbound payment has completed.

#### Request body

```json
{
    "Type": "Target2.FICreditTransfer.Inbound.Completed",
    "Version": 1,                                                         
    "Payload": {...},
    "Nonce":
}
```
