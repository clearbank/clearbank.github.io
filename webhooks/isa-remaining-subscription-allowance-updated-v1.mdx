This webhook triggers when a cash ISA customer's annual allowance is affected by their actions on another cash ISA issued by ClearBank.

<Callout colour="blue">
  This webhook cannot account for customer activity on cash ISAs operating outside of ClearBank's ISA manager license.
</Callout>

#### Request body

```json
{
    "Type":"IsaRemainingSubscriptionAllowanceUpdated",
    "Version": 1,
    "Payload": {...},
    "Nonce":
}
```

#### Payload

| Element                          | Description                                                                                                                                                                                                                           |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `AccountId`                      | The unique identifier for this ISA. <br /> Type: `string` <br /> Format: `GUID`                                                                                                                                                       |
| `TaxYear`                        | The current tax year. <br /> Type: `string` <br /> Pattern: `^[0-9]{2}/[0-9]{2}$`                                                                                                                                                     |
| `CashIsaStatus`                  | The status of this ISA. <br /> Type: `string` <br /> Example: `None`, `Inactive`, `Active`, `Cancelled`, `Closed`, `TransferredOut`                                                                                                   |
| `NetSubscription`                | The net amount withdrawn and deposited to this ISA in the current tax year. This can value can be negative. <br /> Type: `number` <br /> Format: `decimal`                                                                            |
| `RemainingSubscriptionAllowance` | The available amount the customer can deposit in the current tax year to this ISA. Reflects withdrawals and deposits from all of this customer's ISAs issued by ClearBank. <br /> Type: `number` <br /> Format: `decimal` |
| `AnnualAllowanceUsed`            | The total amount that has been deposited into this ISA in the current tax year. <br /> Type: `number` <br /> Format: `decimal`                                                                                                        |

#### Example ISA Remaining Subscription Allowance Updated webhook request body

```json
{
    "Type":"IsaRemainingSubscriptionAllowanceUpdated",
    "Version": 1,
    "Payload":
    {
        "AccountId": "57d56e32-efca-49e7-a71f-8eb18d72c162",
        "TaxYear": "24/25",
        "CashIsaStatus": "Active",
        "NetSubscription": 8000.00,
        "RemainingSubscriptionAllowance": 10000.00,
        "AnnualAllowanceUsed": 8000.00
    },
    "Nonce": 5582946458
}
```

In this example, the customer has deposited £2,000 into a different cash ISA held with ClearBank, decreasing the **RemainingSubscriptionAllowance** from £12,000 to £10,000.

#### Example webhook response

```json
{
    "Nonce": 5582946458
}
```
