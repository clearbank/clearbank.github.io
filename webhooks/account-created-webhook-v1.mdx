This webhook confirms the account has been created.

#### Request body

```json
{
    "Type":"AccountCreated",
    "Version":1,
    "Payload": {...},
    "Nonce":
}
```

<WebhookBlockPayload webhookName='AccountCreated'></WebhookBlockPayload>

#### Payload
| Element              | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|----------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `AccountId`          | Account identifier used to uniquely identify the  account. <br /> Type: `string` <br /> Format: `GUID`                                                                                                                                                         |
| `AccountName`        |  Name of the account. <br /> Type: `string`<br /> Max. length: `128`                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `AccountHolderLabel` | The account description. <br /> Type: `string` <br /> Max. length: `100`                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `AccountIdentifiers` | Account identifier details. <br /> Type: `object`<br /><br /> `IBAN`: International Bank Account Number associated with the account.<br /> Type: `string`  <br /> Max. length: `34` <br /><br />  `BBAN`: Basic Bank Account number associated with the account.<br /> Type: `string` <br /> Max. length: `30` <br />  |
| `TimestampCreated`   | The date and time the account was created.<br /> Type: `string` <br /> Time Zone: `UTC` <br /> Format: `DateTime`                                                                                                                                                                                                                                                                                                                                                              |
| `AccountType`        | Type of account. <br />Type: `string`<br />Examples: `CurrentAccount`, `DepositAccount`, `ControlAccount`, `SegregatedClientAccount`, `InstitutionAccount`                                                                                                                                                                                                                                                                                                                                                |
| `Nonce`              | Cryptographically secure number generated by ClearBankÂ® for every single webhook. A nonce number is generated randomly and should not be used as a check for duplication.                                                                                                                                                                                                                                                                                                                                                                  |

#### Example account created webhook request body

```json
{
    "Type":"AccountCreated",
    "Version":1,
    "Payload":{
        "AccountId":"56de3483-a569-42eb-90cc-acf59661356f",
        "AccountName":"JOHN DOE",
        "AccountHolderLabel":"JOHN DOE",
        "AccountIdentifiers":{
            "IBAN":"GB00CUBK11223312345678",
            "BBAN":"CUBK11223312345678"
        },
        "TimestampCreated":"2018-07-19T13:31:30.8789344Z",
        "AccountType":"CurrentAccount"
    },
    "Nonce":1082937278
}
```

}
