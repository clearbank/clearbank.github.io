This webhook confirms an inbound cross-border payment has completed. It includes detailed information about the payment using standard ISO 20022 fields.

#### Request body

```json
{
    "Type": "Payments.CrossBorder.InboundPaymentCompleted",
    "Version": 1,
    "Payload": {...},
    "Nonce":
}
```
