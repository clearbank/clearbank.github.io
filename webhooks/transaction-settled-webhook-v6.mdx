title: "Transaction Settled Webhook"

**This sends a webhook notification confirming the transaction has settled**

#### Request body

```json
{
    "Type":"TransactionSettled",
    "Version":6,
    "Payload": {...},
    "Nonce":
}
```

#### Payload
| Element                       | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|-------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `TransactionId`               | Unique Identification of the Transaction.<br /> Type: `GUID` <br /> Max. Length: `36`<br /> Special Character: Hyphen `-`<br /> Example format of a typical TransactionId:`CE25B426-9430-4A82-8EB8-9311DA2D3E12`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `Status`                      | Transaction status.<br />Type: `string`<br />`Settled`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `Scheme`                      | *Optional* <br />The Scheme type.<br /> `Transfer`<br />`FasterPayments`<br />`Bacs`<br />`Chaps`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| `EndToEndTransactionId`       | Unique Identifier supplied by you in your API request. For inbound transactions, this identifier may not be unique.<br />Type: `string`<br />Max. length:`35`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| `Amount`                      | Transaction amount. <br />Type: `int`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `TimestampSettled`            | Date and time the transaction settled.<br /> Time Zone: `UTC`<br /> Type: `DateTime`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| `TimestampCreated`            | Date and time the transaction was created.<br /> Time Zone: `UTC`<br /> Type: `DateTime`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| `CurrencyCode`                | *Optional* <br />Code of transaction currency.<br /> `GBP`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| `DebitCreditCode`             | Movement of transaction.<br />Type: `string`<br />Options:`Credit`, `Debit`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `Reference`                   | *(optional)* <br/>Payment transaction reference.<br /> Max. length: `140`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| `IsReturn`                    | Is the Transaction returned from Scheme?<br /> `True`<br/> `False`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| `Account`                     | Account information object<br />Type: `object`<br /> It contains the `IBAN`, `BBAN`, `OwnerName`, `TransactionOwner` and `InstitutionName`.<br /><br />`IBAN`: International Bank Account Number associated with the account.<br /> Type: `string`  <br /> Max. length: `34` <br /><br />  `BBAN`: Basic Bank Account number associated with the account.<br /> Type: `string` <br /> Max. length: `30`<br /><br /> **Depending on Scheme Type**<br /> `OwnerName`: name of the beneficiary account holder registered with ClearBank®.<br />Type: `string`<br /> Max. length: `128`<br /> - Faster Payments = Sender’s Account Name<br /> - Chaps = Ordering Customer’s Name<br /><br /> `TransactionOwnerName`<br />Type: `string`<br />Max. length: `128`<br /> - Beneficiary Account Holder, as specified by the remitter in the payment message.<br /> - For outbound payments, as specified by ClearBank® as the remitter name in the payment message.<br /><br /> `InstitutionName`: The institution name. <br />Max. length: `128`<br /> - Chaps = Ordering Institution                                                                                                                            |
| `CounterpartAccount`          | The account which is not held at ClearBank. <br /> It contains the `IBAN`, `BBAN`, `OwnerName`, `TransactionOwner` and `InstitutionName`.<br /><br />`IBAN`*(Optional)*: International Bank Account Number associated with the account.<br /> Type: `string`  <br /> Max. length: `34` <br /><br />  `BBAN`*(Optional)*: Basic Bank Account number associated with the account.<br /> Type: `string` <br /> Max. length: `30`<br /><br /> **Depending on Scheme Type**<br /> `OwnerName`: name of the beneficiary account.<br />Type: `string`<br /> Max. length: `128`<br /> - Populated with either the same name as `TransactionOwnerName`, or with `“Not Provided”` depending on what is provided in the payment message.<br />- Faster Payments = Beneficiary's Account Name<br /><br /> `TransactionOwnerName`<br />Type: `string`<br />Max. length: `128`<br /> Populated by Scheme with the name provided in the payment message.<br /> - For outbound payments, this is populated as the Beneficiary Name which is provided by you.<br /><br /> `InstitutionName`: The institution name. <br />Max. length: `128`<br /> - Chaps = Beneficiary Institution |
| `ActualEndToEndTransactionId` | Unique payment Identifier depending on Scheme Type.<br /> Maximum Field Length: `255`<br /> Faster Payments = `FPID`<br /> Chaps = `Field 20`<br /><br />  **FPID** is the unique payment identifier generated by Faster Payments. <br /> **Field 20** is the unique payment identifier generated by the remitting bank.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `DirectDebitMandateId`        | *(Optional)*<br />Unique Identification of a Direct Debit Instruction.<br />Type: `GUID`<br /> Max. length: `36`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `TransactionCode`             | *(Optional)*<br/>Direct Debit Transaction Code:<br /> `"01"` - First collection<br /> `"17"` - Regular collection<br /> `"18"` - Re-presented collection<br /> `"19"` - Final collection<br /><br /> *(Optional)*<br/>Direct Credit Transaction Code:<br /> `"99"` - Standard Bank Giro Credit<br /> `"86"` - Bank to bank settlement<br /> `"Z4"` - Interest payment<br /> `"Z5"` - Dividend payment                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| `ServiceUserNumber`           | *(Optional)*<br />Service User's Unique Identifier registered with Bacs.<br /> Max. length: `6`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `BacsTransactionId`           | *(Optional)*<br /> Type: `GUID` <br />Max. length: `36`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| `BacsTransactionDescription`  | *(Optional)*<br /> `CreditTransactionSettled`<br /> `DebitContraTransactionSettled`<br /> `DebitTransactionSettled`<br /> `CreditContraTransactionSettled`<br /> `DirectDebitReceivedIndemnityClaimSettled`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `TransactionType`             | *(Optional)* <br /> `DirectCredit`<br /> `DirectDebit`<br /> `DirectDebitIndemnityClaim`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| `TransactionSource`           | *(Optional)*<br />Origination of the Transaction :<br /> Max. length: `50` <br /> Example: `CardProcessor`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| `SupplementaryData`           | *(Optional)*<br />Transaction supplementary data<br /> It is an array with the a `Name` and `Value`. <br /><br /> `Name`: Field Number/Name for an inbound CHAPS or FPS message. <br />Max. length: `33` <br /> `Name` options: `20`, `13C`, `23B`, `23E`, `26T`, `32A`, `33B`, `36`, `50a`, `51A`, `52a`, `53a`, `54a`, `55a`, `56a`, `57a`, `59a`, `70`, `71A`, `71F`, `71G`, `72`, `77B`, `SendingFpsInstitution`, `Currency`, `DateSent`, `PaymentType`, `TransactionReferenceNumber`, `Amount`, `OriginatingCreditInstitution`, `OriginatingCustomerAccountNumber`, `OriginatingCustomerAccountName`, `OriginatingCustomerAccountAddress`, `EndToEndReference`, `BeneficiaryCreditInstitution`, `BeneficiaryCustomerAccountNumber`, `BeneficiaryCustomerAccountName`, `BeneficiaryCustomerAccountAddress`, `ReturnedPaymentFpId`, `PaymentReturnCode`, `PaymentSubTypeCode`, `PurposeTransactionType`, `TypeOfAccountCode`, `NumericReference`, `ReferenceInformation`, `RemittanceInformation`, `RegulatoryReporting`, `OriginalCurrency`, `OriginalAmount`, `ExchangeRate`, `ChargingInformation`, `SubmittingMember`, `ReceivingMember`, `SettlementCycleId`, `FileId`, `ProcessedAsynchronously`, `AgencyAccountWithMember`, `AgencySortCodeWithMember` <br /><br />    `Value`<br /> Max. length: `500` <br /><br /> **There could be more than one Name and Value pair.**                                                                         |
| `MXMessage`                       | *(Optional)*<br />Only if `Scheme` is Chaps; ISO 20022 (MX) compliant version of the transaction message. *Note*: This value can be very long.<br />Type: `string`<br /> Min. Length: `0`<br /> Max. Length: `80000` <br /> Example: See below.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| `Nonce`                       | Cryptographically secure number generated by ClearBank® for every single webhook. A nonce number is generated randomly and should not be used as a check for duplication.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
