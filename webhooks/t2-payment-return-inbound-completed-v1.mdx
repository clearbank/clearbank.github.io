title: "T2 Payment Return Inbound Completed webhook"

This webhook confirms a T2 return inbound payment has completed.

#### Request body

```json
{
    "Type": "Target2.PaymentReturn.Inbound.Completed",
    "Version": 1,                                                         
    "Payload": {...},
    "Nonce":
}
```
