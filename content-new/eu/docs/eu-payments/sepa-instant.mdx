## SEPA Instant

SEPA CT, also known as the Single Euro Payments Area Credit Transfer, enables you to move Euros across participating countries.

You can send a SEPA CT payment using the [/v1/payments/sepa](#send-a-sepa-ct-payment) endpoint. To recall a payment, use the [/v1/payments/sepa/{paymentId}/recall](#recall-a-sepa-ct-payment) endpoint.

### Incoming SEPA CT recall requests

You may recieve a recall request via the [Recall Request webhook](). To respond to the recall request, use the [/v1/payments/sepa/{paymentId}/recall-response](#respond-to-a-SEPA-CT-payment-recall-request) endpoint.

You may only return the full amount. Partial refunds are not accepted.

<Callout colour="blue">
If you accept the recall, a pacs.004 payment will be automatically initiated. You do not need to use the send a payment endpoint.
</Callout>
