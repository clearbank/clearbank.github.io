## SEPA Instant

SEPA Instant enables you to move Euros across participating countries.

SEPA Instant has a transaction limit of €100,000 per transaction. Payments can be sent at any time, including bank holidays and weekends.

To submit a payment, use the [POST /v1/payments/sepa-instant](#send-a-sepa-instant-payment) endpoint. You will receive the SEPA Instant Outbound Payment Settled webhook and the EU Transaction Settled webhook.

### SEPA Instant recall requests

You can send SEPA Instant recall requests using the [Request a SEPA Instant recall](request-a-sepa-instant-recall) endpoint. Depending on the reason code, 

You may recieve a recall request via the [SEPA Instant Recall Request webhook](). To respond to the recall request, use the [/v1/payments/sepa-instant/{paymentId}/recall-response](#) endpoint.

You may only return the full amount. Partial refunds are not accepted.

<Callout colour="blue">
If you accept the recall, a pacs.004 payment will be automatically initiated. You do not need to use the send a payment endpoint.
</Callout>
